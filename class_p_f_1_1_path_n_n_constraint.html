详细描述

A special NNConstraint which can use different logic for the start node and end node in a path. A PathNNConstraint can be assigned to the Path.nnConstraint field, the path will first search for the start node, then it will call SetStart and proceed with searching for the end node (nodes in the case of a MultiTargetPath).
The default PathNNConstraint will constrain the end point to lie inside the same area as the start point.
