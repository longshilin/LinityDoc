详细描述

Stores temporary node data for a single pathfinding request. Every node has one PathNode per thread used. It stores e.g G score, H score and other temporary variables needed for path calculation, but which are not part of the graph structure.

ushort PF.PathNode.heapIndex = BinaryHeap.NotInHeap

Index of the node in the binary heap. The open list in the A* algorithm is backed by a binary heap. To support fast 'decrease key' operations, the index of the node is saved here.
