◆ GetAreaColor()

static Color Pathfinding.AstarColor.GetAreaColor ( uint  area)
static

Returns an color for an area, uses both user set ones and calculated. If the user has set a color for the area, it is used, but otherwise the color is calculated using Mathfx.IntToColor

◆ OnEnable()

void Pathfinding.AstarColor.OnEnable ( )

Pushes all local variables out to static ones. This is done because that makes it so much easier to access the colors during Gizmo rendering and it has a positive performance impact as well (gizmo rendering is hot code).
