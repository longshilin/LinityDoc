详细描述

Exposes internal methods for graphs. This is used to hide methods that should not be used by any user code but still have to be 'public' or 'internal' (which is pretty much the same as 'public' as this library is distributed with source code).

Hiding the internal methods cleans up the documentation and IntelliSense suggestions.
