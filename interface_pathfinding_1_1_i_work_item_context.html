◆ QueueFloodFill()

void Pathfinding.IWorkItemContext.QueueFloodFill ( )

Call during work items to queue a flood fill. An instant flood fill can be done via FloodFill() but this method can be used to batch several updates into one to increase performance. WorkItems which require a valid Flood Fill in their execution can call EnsureValidFloodFill to ensure that a flood fill is done if any earlier work items queued one.

Once a flood fill is queued it will be done after all WorkItems have been executed.
